# React Todo-List

This project is a simple Todo-List application built with React. It's currently in progress and has the basic functionality of a Todo-List.

## Features

- Tasks can be saved in two formats: Important tasks and Regular tasks.
- Each task has three options: Done, Delete, and Edit.
- The application is user-friendly and intuitive to use.

## Implementation Details

- The application uses browser storage for saving data, ensuring that your tasks persist even after the browser is closed.
- State management is handled using React Context, providing a clean and efficient way to manage global state without the need to pass props down through multiple layers.

## Future Improvements

The project is still in progress, and more features will be added in the future. Stay tuned!

## Getting Started

To get started with this project:

1. Clone the repository
2. Install the dependencies with `npm install`
3. Start the development server with `npm start`

Now you can open your browser to `http://localhost:3000` and start using the application!
